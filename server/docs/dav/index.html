WebDAV/CalDAV/CardDAV Protocol Documentation

HTTP extension methods for calendar and contact synchronization

CalDAV

Calendar synchronization protocol (RFC 4791). Used by Apple Calendar, Thunderbird, and other clients.

Endpoint: /dav/calendars/{username}/

CardDAV

Contact synchronization protocol (RFC 6352). Used by Apple Contacts, Thunderbird, and other clients.

Endpoint: /dav/addressbooks/{username}/

WebDAV Methods

PROPFIND Retrieve properties of a resource (collection or item)
PROPPATCH Update properties of a resource
MKCALENDAR Create a new calendar collection
MKCOL Create a new collection (address book)
REPORT Query calendar/contact data (calendar-query, sync-collection)
GET Retrieve a resource (event .ics, contact .vcf)
PUT Create or update a resource
DELETE Remove a resource

Authentication

DAV endpoints use HTTP Basic Authentication with either:

  • App Passwords (created in account settings)
  • CalDAV Credentials (calendar-specific access)
  • CardDAV Credentials (address book-specific access)

Example: PROPFIND Calendar List

curl -X PROPFIND https://server/dav/calendars/johndoe/ \
  -H "Authorization: Basic am9obmRvZTpwYXNzd29yZA==" \
  -H "Content-Type: application/xml" \
  -H "Depth: 1" \
  -d '<?xml version="1.0" encoding="UTF-8"?>
<propfind xmlns="DAV:">
  <prop>
    <displayname/>
    <resourcetype/>
    <getctag xmlns="http://calendarserver.org/ns/"/>
  </prop>
</propfind>'

Example: sync-collection REPORT

curl -X REPORT https://server/dav/calendars/johndoe/personal/ \
  -H "Authorization: Basic am9obmRvZTpwYXNzd29yZA==" \
  -H "Content-Type: application/xml" \
  -d '<?xml version="1.0" encoding="UTF-8"?>
<sync-collection xmlns="DAV:">
  <sync-token/>
  <sync-level>1</sync-level>
  <prop>
    <getetag/>
    <calendar-data xmlns="urn:ietf:params:xml:ns:caldav"/>
  </prop>
</sync-collection>'

XML Namespaces

DAV: WebDAV core namespace
urn:ietf:params:xml:ns:caldav CalDAV namespace
urn:ietf:params:xml:ns:carddav CardDAV namespace
http://calendarserver.org/ns/ Apple Calendar Server extensions (getctag)
